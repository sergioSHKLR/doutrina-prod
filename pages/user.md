# 👤 Leitor

Aqui você encontra os passos essenciais para aproveitar ao máximo o nosso projeto.

## Guia Rápido

### Visibilidade de Colunas

A interface se adapta automaticamente ao tamanho da tela. Suas preferências tipográficas ficam salvas automaticamente.

![](../assets/images/doutrina-demo.gif)

**Colunas da interface:**

1. **Menu** — livros, partes e capítulos  
2. **Principal** — texto adaptado com links  
3. **Complementar** — artigos, definições, mapas, vídeos, etc.  
4. **Anotações** — grifos e anotações pessoais

### Acesso ao conteúdo

**a.** Escolha um livro — No Menu lateral, clique no livro desejado.  

**b.** Navegue pelo conteúdo — No mesmo Menu lateral, selecione parte ou capítulo. A página rola automaticamente.  

**c.** Use os links — No painel Principal, clique em termos com link. O conteúdo abre no painel Complementar.  

**d.** Anotações pessoais — Use a aba de Anotações para grifar trechos e escrever suas notas (via Hypothes.is).
