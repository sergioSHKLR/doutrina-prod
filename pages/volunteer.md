# 🛠️ Voluntário

Este é um projeto **amador** em sua essência, desenvolvido por uma pessoa com background técnico porém sem formação profissional na àrea. **Qualquer ajuda profissional é extremamente bem-vinda** e valorizada.

Se você tem experiência em desenvolvimento web, design, Jekyll, JavaScript, ou acessibilidade, sua contribuição pode fazer uma grande diferença.

## Como você pode ajudar

- **Melhoria de links** — Sugerir ou corrigir links para artigos, definições e vídeos
- **Revisão de texto** — Correções ortográficas, gramaticais ou de formatação
- **Novos recursos** — Ideias de funcionalidades, melhorias na interface ou funcionalidade
- **Tradução** — Traduzir a interface para outras línguas
- **Testes** — Testar em diferentes dispositivos e relatar problemas
- **Divulgação** — Compartilhar o projeto com quem possa se beneficiar

### Tarefas recomendadas para iniciantes

1. Leia o **[Guia Rápido](./user.html)**
2. Leia o **[Manual Avançado](./advanced-manual.html)**
3. Explorar o site e identificar pontos de melhoria
4. Abrir uma Issue no GitHub com sua sugestão ou erro encontrado

### Como contribuir tecnicamente

- O repositório está aberto no GitHub
- Você pode fazer fork, editar os arquivos Markdown e abrir Pull Requests
- Qualquer ajuda é valorizada, desde pequenas correções até grandes melhorias
